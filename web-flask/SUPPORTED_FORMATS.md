# 文档上传支持的文件格式

## 当前支持的格式

✅ **DOCX** - Microsoft Word 2007+ 格式 (.docx)
- 完全支持内容提取
- 支持段落和表格内容
- 自动生成摘要

✅ **PDF** - 便携式文档格式 (.pdf)
- 支持文本内容提取
- 使用 pdfplumber 库处理

✅ **TXT** - 纯文本格式 (.txt)
- 支持 UTF-8 和 GBK 编码
- 自动清理格式字符

## 不支持的格式

❌ **DOC** - Microsoft Word 97-2003 格式 (.doc)
- 原因：python-docx 库不支持旧版 .doc 格式
- 解决方案：请将文件另存为 .docx 格式

❌ 其他格式（如：.jpg, .png, .xls, .ppt 等）

## 使用建议

1. **推荐使用 DOCX 格式**：保持最佳兼容性和功能支持
2. **文件大小**：建议单个文件不超过 50MB
3. **文件命名**：使用中文或英文文件名，避免特殊字符
4. **内容质量**：确保文档内容清晰，避免扫描版PDF

## 技术依赖

- `python-docx==1.2.0` - DOCX文件处理
- `pdfplumber==0.11.7` - PDF文件处理
- `Flask==3.0.3` - Web框架

## 错误处理

如果上传失败，请检查：
1. 文件格式是否在支持列表中
2. 文件是否损坏
3. 文件大小是否合理
4. 网络连接是否正常
